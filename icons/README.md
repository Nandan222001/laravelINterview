# PWA Icons

This directory contains the icon files for the Progressive Web App.

## Icon Sizes

The following icon sizes are required for PWA:

- 72x72px
- 96x96px
- 128x128px
- 144x144px
- 152x152px
- 192x192px
- 384x384px
- 512x512px

## Generating Icons

1. Open `generate-icons.html` in a browser
2. Click "Generate All Icons" to preview
3. Click "Download All Icons" to save all icon files
4. Place the downloaded icons in this directory

## Manual Creation

If you prefer to create icons manually:

1. Create a square image with the Laravel Interview Bank logo
2. Export in the sizes listed above
3. Name them as: `icon-{size}x{size}.png`
4. Ensure transparency is supported
5. Use the app's theme color (#e3342f) as the primary color

## Icon Guidelines

- Use a simple, recognizable design
- Ensure the icon is visible at small sizes
- Follow the app's color scheme (red gradient)
- Test on both light and dark backgrounds
- Icons should be optimized PNG files
