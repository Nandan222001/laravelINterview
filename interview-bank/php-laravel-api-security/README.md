# PHP Laravel API Security

Advanced interview questions covering Laravel security patterns, authentication, authorization, API protection, and secure coding practices.

## 📌 Topics Covered

- **Authentication & Authorization**: JWT, Passport, Sanctum, OAuth2
- **API Security**: Rate limiting, CORS, CSRF protection, XSS prevention
- **Data Protection**: Encryption, hashing, secure data storage
- **Middleware & Guards**: Custom authentication guards, policy-based authorization
- **Vulnerability Prevention**: SQL injection, mass assignment, insecure deserialization
- **Security Headers**: CSP, HSTS, X-Frame-Options
- **Secure Coding**: Input validation, output encoding, dependency management

## 🎯 Question Categories

### Authentication Mechanisms
Questions focusing on implementing secure authentication systems using Laravel's built-in tools and third-party packages.

### API Rate Limiting & Throttling
Advanced rate limiting strategies for protecting APIs from abuse and ensuring fair usage.

### Authorization & Permissions
Role-based access control (RBAC), policy-based permissions, and fine-grained authorization.

### Cryptography & Data Protection
Implementing encryption, secure key management, and protecting sensitive data at rest and in transit.
