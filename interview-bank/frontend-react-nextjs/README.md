# Frontend React & Next.js

Interview questions covering modern React patterns, Next.js architecture, state management, server-side rendering, and frontend performance optimization.

## 📌 Topics Covered

- **React Patterns**: Hooks, context, composition, render props, HOCs
- **Next.js Features**: SSR, SSG, ISR, API routes, middleware
- **State Management**: Redux, Zustand, Jotai, React Query, SWR
- **Performance**: Code splitting, lazy loading, memoization, virtual scrolling
- **Routing**: Dynamic routes, nested layouts, parallel routes, intercepting routes
- **Data Fetching**: Server components, client components, streaming
- **TypeScript**: Type safety, generics, utility types, strict mode

## 🎯 Question Categories

### React Architecture
Component design, composition patterns, and building scalable React applications.

### Next.js Rendering Strategies
Understanding SSR, SSG, ISR, and choosing the right rendering strategy for different scenarios.

### State Management
Implementing efficient state management solutions for complex applications.

### Performance Optimization
Techniques for optimizing bundle size, render performance, and user experience.
