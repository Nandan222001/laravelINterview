1. Variables & Data Types

Variable Declaration and Naming

PHP is a loosely typed language where variables don't require explicit type declarations. Variables start with $ and are case-sensitive.

<?php
// Variable declaration
$name = "John Doe";              // String
$age = 30;                       // Integer
$height = 5.9;                   // Float
$isActive = true;                // Boolean
$data = null;                    // NULL
$fruits = ["apple", "banana"];   // Array

// Naming conventions:
// - Must start with $ followed by letter or underscore
// - Can contain letters, numbers, and underscores
// - Case-sensitive: $name !== $Name

$validName = "John";
$_privateVar = "hidden";
$user2 = "Jane";

// Invalid names:
// $2user = "error";  // Cannot start with number
// $user-name = "error"; // Cannot use hyphens
?>

Data Types in PHP

Type Description Example
String Sequence of characters "Hello World"
Integer Whole numbers 42, -17, 0
Float Decimal numbers 3.14, -0.5
Boolean True or false true, false
Array Collection of values [1, 2, 3]
Object Instance of a class new User()
NULL No value null
Resource External resource reference Database connection

Type Juggling and Type Casting

<?php
// Type juggling (automatic conversion)
$result = "10" + 5;  // 15 (string converted to int)
$concat = "10" . 5;  // "105" (int converted to string)

// Explicit type casting
$str = "123";
$int = (int)$str;        // Cast to integer: 123
$float = (float)"3.14";  // Cast to float: 3.14
$bool = (bool)1;         // Cast to boolean: true
$array = (array)$str;    // Cast to array: ["123"]

// Type checking functions
var_dump(is_string($str));      // bool(true)
var_dump(is_int($int));         // bool(true)
var_dump(is_float($float));     // bool(true)
var_dump(is_bool($bool));       // bool(true)
var_dump(is_array($array));     // bool(true)
var_dump(is_null(null));        // bool(true)
var_dump(is_numeric("123"));    // bool(true)

// Get variable type
echo gettype($int);  // "integer"
echo get_debug_type($int);  // "int" (PHP 8+)
?>

Variable Scope

<?php
// Global scope
$globalVar = "I'm global";

function testScope() {
    // Local scope
    $localVar = "I'm local";
    
    // Access global variable
    global $globalVar;
    echo $globalVar; // "I'm global"
    
    // Alternative: $GLOBALS superglobal
    echo $GLOBALS['globalVar'];
    
    // Static variables persist between function calls
    static $counter = 0;
    $counter++;
    echo "Counter: $counter";
}

testScope();  // Counter: 1
testScope();  // Counter: 2

// Variable variables
$var = "hello";
$$var = "world";  // Creates $hello = "world"
echo $hello;      // "world"
?>
💡 Best Practice: Use type hints and return type declarations in PHP 7+ for better code reliability and IDE support. Minimize use of global variables.

2. Operators

Arithmetic Operators

<?php
$a = 10;
$b = 3;

echo $a + $b;   // 13 (Addition)
echo $a - $b;   // 7  (Subtraction)
echo $a * $b;   // 30 (Multiplication)
echo $a / $b;   // 3.333... (Division)
echo $a % $b;   // 1  (Modulus - remainder)
echo $a ** $b;  // 1000 (Exponentiation - PHP 5.6+)

// Integer division (PHP 7+)
echo intdiv(10, 3);  // 3

// Increment/Decrement
$x = 5;
echo ++$x;  // 6 (Pre-increment: increment then return)
echo $x++;  // 6 (Post-increment: return then increment, $x is now 7)
echo --$x;  // 6 (Pre-decrement)
echo $x--;  // 6 (Post-decrement, $x is now 5)
?>

Comparison Operators

<?php
// Equal (loose comparison - type juggling)
var_dump(5 == "5");   // bool(true)
var_dump(0 == false); // bool(true)

// Identical (strict comparison - no type juggling)
var_dump(5 === "5");  // bool(false)
var_dump(5 === 5);    // bool(true)

// Not equal
var_dump(5 != 6);     // bool(true)
var_dump(5 <> 6);     // bool(true) - alternative syntax

// Not identical
var_dump(5 !== "5");  // bool(true)

// Greater than, less than
var_dump(5 > 3);      // bool(true)
var_dump(5 < 3);      // bool(false)
var_dump(5 >= 5);     // bool(true)
var_dump(5 <= 5);     // bool(true)

// Spaceship operator (PHP 7+)
// Returns -1, 0, or 1
echo 1 <=> 1;  // 0 (equal)
echo 1 <=> 2;  // -1 (less than)
echo 2 <=> 1;  // 1 (greater than)

// Null coalescing operator (PHP 7+)
$username = $_GET['user'] ?? 'guest';  // Use 'guest' if not set

// Null coalescing assignment (PHP 7.4+)
$config['timeout'] ??= 30;  // Assign only if null or not set
?>

Logical Operators

<?php
$a = true;
$b = false;

// AND operators
var_dump($a && $b);  // bool(false) - higher precedence
var_dump($a and $b); // bool(false) - lower precedence

// OR operators
var_dump($a || $b);  // bool(true) - higher precedence
var_dump($a or $b);  // bool(true) - lower precedence

// NOT operator
var_dump(!$a);       // bool(false)

// XOR operator (true if exactly one is true)
var_dump($a xor $b); // bool(true)
var_dump(true xor true); // bool(false)

// Short-circuit evaluation
$result = $a || expensiveFunction();  // expensiveFunction() not called
$result = $b && expensiveFunction();  // expensiveFunction() not called
?>

String Operators

<?php
// Concatenation
$first = "Hello";
$last = "World";
$full = $first . " " . $last;  // "Hello World"

// Concatenation assignment
$text = "Hello";
$text .= " World";  // "Hello World"

// String interpolation
$name = "John";
echo "Hello, $name!";           // "Hello, John!"
echo "Hello, {$name}!";         // "Hello, John!"
echo "Hello, {$arr['key']}!";  // Access array in string
?>
✅ Remember: Always use strict comparison (=== and !==) unless you specifically need type juggling. Be aware of operator precedence.

3. Control Structures

If-Else Statements

<?php
$score = 85;

// Basic if-else
if ($score >= 90) {
    echo "Grade: A";
} elseif ($score >= 80) {
    echo "Grade: B";
} elseif ($score >= 70) {
    echo "Grade: C";
} elseif ($score >= 60) {
    echo "Grade: D";
} else {
    echo "Grade: F";
}

// Alternative syntax (useful in templates)
if ($loggedIn):
    echo "Welcome back!";
else:
    echo "Please login";
endif;

// Match expression (PHP 8+)
$result = match(true) {
    $score >= 90 => 'A',
    $score >= 80 => 'B',
    $score >= 70 => 'C',
    $score >= 60 => 'D',
    default => 'F'
};
?>

Switch Statements

<?php
$day = "Monday";

switch ($day) {
    case "Monday":
    case "Tuesday":
    case "Wednesday":
    case "Thursday":
    case "Friday":
        echo "Weekday";
        break;
    case "Saturday":
    case "Sunday":
        echo "Weekend";
        break;
    default:
        echo "Invalid day";
}

// Switch uses loose comparison (==)
$value = "1";
switch ($value) {
    case 1:  // This matches because "1" == 1
        echo "Matched";
        break;
}
?>

Loops

<?php
// For loop
for ($i = 0; $i < 5; $i++) {
    echo "Iteration: $i\n";
}

// While loop
$count = 0;
while ($count < 5) {
    echo "Count: $count\n";
    $count++;
}

// Do-while loop (executes at least once)
$num = 0;
do {
    echo "Number: $num\n";
    $num++;
} while ($num < 5);

// Foreach loop
$fruits = ["apple", "banana", "orange"];
foreach ($fruits as $fruit) {
    echo $fruit . "\n";
}

// Foreach with key-value
$person = ["name" => "John", "age" => 30, "city" => "NYC"];
foreach ($person as $key => $value) {
    echo "$key: $value\n";
}

// Modifying array values by reference
foreach ($fruits as &$fruit) {
    $fruit = strtoupper($fruit);
}
unset($fruit);  // Important: unset reference after loop

// Loop control
for ($i = 0; $i < 10; $i++) {
    if ($i === 3) {
        continue;  // Skip iteration
    }
    if ($i === 8) {
        break;  // Exit loop
    }
    echo $i;
}
?>

4. Functions

Function Declaration and Usage

<?php
// Basic function
function greet($name) {
    return "Hello, $name!";
}

echo greet("John");  // "Hello, John!"

// Function with default parameters
function createUser($name, $role = "user", $active = true) {
    return [
        'name' => $name,
        'role' => $role,
        'active' => $active
    ];
}

$user1 = createUser("John");  // Uses defaults
$user2 = createUser("Jane", "admin");  // Override role

// Type hints (PHP 7+)
function add(int $a, int $b): int {
    return $a + $b;
}

// Nullable types (PHP 7.1+)
function findUser(?int $id): ?array {
    if ($id === null) {
        return null;
    }
    return ['id' => $id, 'name' => 'John'];
}

// Union types (PHP 8+)
function process(int|float $number): int|float {
    return $number * 2;
}

// Void return type
function logMessage(string $msg): void {
    echo $msg;
}
?>

Anonymous Functions and Arrow Functions

<?php
// Anonymous functions (closures)
$greet = function($name) {
    return "Hello, $name!";
};

echo $greet("John");  // "Hello, John!"

// Closures with use keyword
$message = "Welcome";
$greetUser = function($name) use ($message) {
    return "$message, $name!";
};

echo $greetUser("Jane");  // "Welcome, Jane!"

// Modify external variable by reference
$counter = 0;
$increment = function() use (&$counter) {
    $counter++;
};
$increment();
echo $counter;  // 1

// Arrow functions (PHP 7.4+)
$multiply = fn($a, $b) => $a * $b;
echo $multiply(5, 3);  // 15

// Arrow functions inherit parent scope automatically
$factor = 10;
$scale = fn($n) => $n * $factor;
echo $scale(5);  // 50

// Passing to array functions
$numbers = [1, 2, 3, 4, 5];
$squared = array_map(fn($n) => $n ** 2, $numbers);
print_r($squared);  // [1, 4, 9, 16, 25]
?>

Variable Arguments and Named Arguments

<?php
// Variadic functions (PHP 5.6+)
function sum(...$numbers) {
    return array_sum($numbers);
}

echo sum(1, 2, 3, 4, 5);  // 15

// Argument unpacking
function multiply($a, $b, $c) {
    return $a * $b * $c;
}

$numbers = [2, 3, 4];
echo multiply(...$numbers);  // 24

// Named arguments (PHP 8+)
function createPost($title, $content, $published = false) {
    return compact('title', 'content', 'published');
}

// Can specify arguments by name in any order
$post = createPost(
    content: "This is the content",
    title: "My Post",
    published: true
);
?>
⚠️ Caution: Pass by reference can make code harder to understand. Use it sparingly and document when you do.

5. Arrays

Array Creation and Access

<?php
// Indexed arrays
$fruits = ["apple", "banana", "orange"];
$numbers = array(1, 2, 3, 4, 5);  // Old syntax

echo $fruits[0];  // "apple"
echo $numbers[2]; // 3

// Associative arrays
$person = [
    "name" => "John Doe",
    "age" => 30,
    "email" => "john@example.com"
];

echo $person["name"];  // "John Doe"

// Multi-dimensional arrays
$users = [
    [
        "name" => "John",
        "email" => "john@example.com",
        "roles" => ["admin", "editor"]
    ],
    [
        "name" => "Jane",
        "email" => "jane@example.com",
        "roles" => ["user"]
    ]
];

echo $users[0]["name"];      // "John"
echo $users[0]["roles"][0];  // "admin"

// Array destructuring (PHP 7.1+)
[$first, $second, $third] = $fruits;
echo $first;  // "apple"

// Associative array destructuring
["name" => $name, "age" => $age] = $person;
echo $name;  // "John Doe"
?>

Array Manipulation

<?php
$arr = [1, 2, 3];

// Adding elements
$arr[] = 4;              // Append to end: [1, 2, 3, 4]
array_push($arr, 5, 6);  // Append multiple: [1, 2, 3, 4, 5, 6]
array_unshift($arr, 0);  // Prepend: [0, 1, 2, 3, 4, 5, 6]

// Removing elements
$last = array_pop($arr);     // Remove and return last
$first = array_shift($arr);  // Remove and return first
unset($arr[2]);              // Remove by index

// Merging arrays
$arr1 = [1, 2, 3];
$arr2 = [4, 5, 6];
$merged = array_merge($arr1, $arr2);  // [1, 2, 3, 4, 5, 6]
$combined = [...$arr1, ...$arr2];     // Spread operator (PHP 7.4+)

// Searching
$key = array_search("banana", $fruits);  // Returns key if found
$exists = in_array("apple", $fruits);    // Returns boolean
$hasKey = array_key_exists("name", $person);

// Sorting
$numbers = [3, 1, 4, 1, 5, 9, 2, 6];
sort($numbers);      // Sort values ascending (reindex)
rsort($numbers);     // Sort values descending (reindex)
asort($numbers);     // Sort values ascending (maintain keys)
ksort($numbers);     // Sort by keys ascending
usort($numbers, fn($a, $b) => $a <=> $b);  // Custom sort
?>

Array Functions

<?php
$numbers = [1, 2, 3, 4, 5];

// Map: Transform each element
$squared = array_map(fn($n) => $n ** 2, $numbers);
// [1, 4, 9, 16, 25]

// Filter: Keep elements matching condition
$evens = array_filter($numbers, fn($n) => $n % 2 === 0);
// [2, 4]

// Reduce: Combine all elements into single value
$sum = array_reduce($numbers, fn($carry, $n) => $carry + $n, 0);
// 15

// Combining arrays
$keys = ['a', 'b', 'c'];
$values = [1, 2, 3];
$combined = array_combine($keys, $values);
// ['a' => 1, 'b' => 2, 'c' => 3]

// Unique values
$unique = array_unique([1, 2, 2, 3, 3, 3]);
// [1, 2, 3]

// Extract specific column
$users = [
    ['name' => 'John', 'age' => 30],
    ['name' => 'Jane', 'age' => 25]
];
$names = array_column($users, 'name');
// ['John', 'Jane']

// Array info
count($numbers);      // Number of elements
empty($numbers);      // Check if empty
is_array($numbers);   // Type check
?>

6. Error Handling

Try-Catch-Finally

<?php
// Basic try-catch
try {
    $result = riskyOperation();
    echo "Success: $result";
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
    error_log($e->getMessage());
}

// Multiple catch blocks
try {
    $file = new SplFileObject('nonexistent.txt');
} catch (RuntimeException $e) {
    echo "Runtime error: " . $e->getMessage();
} catch (Exception $e) {
    echo "General error: " . $e->getMessage();
}

// Catch multiple exception types (PHP 7.1+)
try {
    // code
} catch (InvalidArgumentException | LogicException $e) {
    echo "Argument or logic error: " . $e->getMessage();
}

// Finally block (always executes)
try {
    $file = fopen('data.txt', 'r');
    $content = fread($file, 1024);
} catch (Exception $e) {
    echo "Error reading file";
} finally {
    // Cleanup code runs regardless of exception
    if (isset($file) && $file) {
        fclose($file);
    }
}
?>

Custom Exceptions

<?php
// Custom exception class
class ValidationException extends Exception {
    private array $errors;
    
    public function __construct(array $errors, string $message = "Validation failed") {
        parent::__construct($message);
        $this->errors = $errors;
    }
    
    public function getErrors(): array {
        return $this->errors;
    }
}

// Using custom exceptions
function processPayment(array $data): void {
    $errors = [];
    
    if (empty($data['amount'])) {
        $errors['amount'] = 'Amount is required';
    }
    
    if (empty($data['card'])) {
        $errors['card'] = 'Card number is required';
    }
    
    if (!empty($errors)) {
        throw new ValidationException($errors);
    }
}

// Handling custom exceptions
try {
    processPayment(['amount' => 100]);
} catch (ValidationException $e) {
    echo "Validation failed:\n";
    foreach ($e->getErrors() as $field => $error) {
        echo "$field: $error\n";
    }
}
?>
💡 Best Practice: Always use specific exception types and provide meaningful error messages. Create custom exceptions for domain-specific errors.

7. Error Types (Parse/Fatal/Warning/Notice)

Understanding PHP Error Types

Error Type Description Stops Execution?
E_PARSE Compile-time syntax errors Yes
E_ERROR (Fatal) Fatal runtime errors Yes
E_WARNING Non-fatal runtime warnings No
E_NOTICE Runtime notices (minor issues) No
E_STRICT Code improvement suggestions No
E_DEPRECATED Deprecated feature warnings No

Parse Errors (E_PARSE)

Parse errors occur during compilation, before execution begins. These are syntax errors that prevent the script from running.

<?php
// Missing semicolon
echo "Hello"
echo "World";  // Parse error: syntax error, unexpected 'echo'

// Unclosed string
$name = "John;  // Parse error: unexpected end of file

// Mismatched brackets
if ($x > 0) {
    echo "positive";
// Parse error: unexpected end of file (missing closing brace)

// Common causes:
// - Missing semicolons
// - Unclosed quotes, brackets, or parentheses
// - Invalid syntax
// - Typos in keywords
?>

Fatal Errors (E_ERROR)

Fatal errors stop script execution immediately. These are serious runtime errors.

<?php
// Calling undefined function
nonExistentFunction();  // Fatal error: Call to undefined function

// Accessing undefined class
$obj = new NonExistentClass();  // Fatal error: Class not found

// Memory exhausted
$array = [];
while (true) {
    $array[] = str_repeat('x', 1000000);
}  // Fatal error: Allowed memory size exhausted

// Maximum execution time exceeded
set_time_limit(1);
while (true) {
    // infinite loop
}  // Fatal error: Maximum execution time exceeded

// Cannot redeclare function
function test() {}
function test() {}  // Fatal error: Cannot redeclare function
?>

Warnings (E_WARNING)

Warnings don't stop execution but indicate problems that should be addressed.

<?php
// Include missing file
include 'nonexistent.php';  // Warning: include(): Failed opening file
echo "Script continues...";  // This still executes

// Division by zero
$result = 10 / 0;  // Warning: Division by zero

// Invalid file operations
$file = fopen('nonexistent.txt', 'r');  // Warning: fopen(): failed to open stream

// Wrong parameter count
function test($a, $b) {}
test(1);  // Warning: Missing argument 2
?>

Notices (E_NOTICE)

Notices indicate minor issues. Script continues running, but these should be fixed.

<?php
// Undefined variable
echo $undefinedVar;  // Notice: Undefined variable

// Undefined array index
$array = ['a' => 1];
echo $array['b'];  // Notice: Undefined index: b

// Use of undefined constant
echo UNDEFINED_CONSTANT;  // Notice: Use of undefined constant

// Accessing property on non-object
$var = null;
echo $var->property;  // Notice: Trying to get property of non-object
?>

Error Reporting Configuration

<?php
// Display all errors (development)
error_reporting(E_ALL);
ini_set('display_errors', 1);

// Hide errors (production)
error_reporting(E_ALL);
ini_set('display_errors', 0);
ini_set('log_errors', 1);
ini_set('error_log', '/path/to/error.log');

// Custom error levels
error_reporting(E_ALL & ~E_NOTICE);  // All except notices
error_reporting(E_ERROR | E_WARNING);  // Only errors and warnings

// Custom error handler
set_error_handler(function($errno, $errstr, $errfile, $errline) {
    $message = "[$errno] $errstr in $errfile on line $errline";
    error_log($message);
    
    // Convert errors to exceptions
    if (error_reporting() & $errno) {
        throw new ErrorException($errstr, 0, $errno, $errfile, $errline);
    }
    
    return true;  // Don't execute PHP internal error handler
});

// Trigger custom errors
trigger_error("Custom warning message", E_USER_WARNING);
trigger_error("Custom notice message", E_USER_NOTICE);
trigger_error("Custom error message", E_USER_ERROR);  // Fatal
?>
⚠️ Important: In development, show all errors (E_ALL). In production, log errors but don't display them to users. Never use @ operator to hide errors - always handle them properly.

8. Magic Methods

Constructor and Destructor (__construct, __destruct)

<?php
class User {
    private string $name;
    
    // __construct: Called when creating new object
    public function __construct(string $name) {
        $this->name = $name;
        echo "User created: $name\n";
    }
    
    // __destruct: Called when object is destroyed
    public function __destruct() {
        echo "User destroyed: {$this->name}\n";
        // Cleanup: close connections, save data, etc.
    }
}

$user = new User("John");  // "User created: John"
unset($user);  // "User destroyed: John"

// Constructor property promotion (PHP 8+)
class Product {
    public function __construct(
        public string $name,
        public float $price,
        private int $stock = 0
    ) {
        // Properties automatically created and assigned
    }
}

$product = new Product("Laptop", 999.99);
echo $product->name;  // "Laptop"
?>

Property Overloading (__get, __set, __isset, __unset)

<?php
class MagicProperties {
    private array $data = [];
    
    // __get: Called when accessing inaccessible property
    public function __get(string $name) {
        echo "Getting property: $name\n";
        return $this->data[$name] ?? null;
    }
    
    // __set: Called when setting inaccessible property
    public function __set(string $name, $value): void {
        echo "Setting property: $name = $value\n";
        $this->data[$name] = $value;
    }
    
    // __isset: Called when checking if inaccessible property is set
    public function __isset(string $name): bool {
        return isset($this->data[$name]);
    }
    
    // __unset: Called when unsetting inaccessible property
    public function __unset(string $name): void {
        unset($this->data[$name]);
    }
}

$obj = new MagicProperties();
$obj->name = "John";      // Calls __set
echo $obj->name;          // Calls __get
if (isset($obj->name)) {} // Calls __isset
unset($obj->name);        // Calls __unset
?>

Method Overloading (__call, __callStatic)

<?php
class MagicMethods {
    private array $data = [];
    
    // __call: Called when invoking inaccessible instance method
    public function __call(string $name, array $arguments) {
        echo "Called method: $name\n";
        
        // Example: magic getters/setters
        if (strpos($name, 'get') === 0) {
            $property = lcfirst(substr($name, 3));
            return $this->data[$property] ?? null;
        }
        
        if (strpos($name, 'set') === 0) {
            $property = lcfirst(substr($name, 3));
            $this->data[$property] = $arguments[0] ?? null;
            return $this;
        }
    }
    
    // __callStatic: Called when invoking inaccessible static method
    public static function __callStatic(string $name, array $arguments) {
        echo "Called static method: $name\n";
    }
}

$obj = new MagicMethods();
$obj->setName("John");    // Calls __call
echo $obj->getName();     // Calls __call
MagicMethods::staticMethod("test");  // Calls __callStatic
?>

Other Important Magic Methods

<?php
class MagicExample {
    private array $data = ['name' => 'John', 'age' => 30];
    
    // __toString: Convert object to string
    public function __toString(): string {
        return json_encode($this->data);
    }
    
    // __invoke: Allow object to be called as function
    public function __invoke(...$args) {
        return "Called with: " . implode(', ', $args);
    }
    
    // __clone: Called when object is cloned
    public function __clone() {
        // Deep copy of reference types
        $this->data = array_merge([], $this->data);
    }
    
    // __sleep: Called before serialize()
    // Return array of property names to serialize
    public function __sleep(): array {
        return ['data'];  // Only serialize $data property
    }
    
    // __wakeup: Called after unserialize()
    public function __wakeup(): void {
        // Restore connections, resources, etc.
    }
    
    // __debugInfo: Customize var_dump() output
    public function __debugInfo(): array {
        return [
            'name' => $this->data['name'],
            'age' => $this->data['age'],
            'info' => 'Custom debug info'
        ];
    }
}

// Using magic methods
$obj = new MagicExample();

echo $obj;  // __toString: {"name":"John","age":30}
$result = $obj(1, 2, 3);  // __invoke
$clone = clone $obj;  // __clone
$serialized = serialize($obj);  // __sleep
$restored = unserialize($serialized);  // __wakeup
var_dump($obj);  // __debugInfo
?>
💡 Magic Methods Summary:
  • __construct, __destruct: Object lifecycle
  • __get, __set, __isset, __unset: Property overloading
  • __call, __callStatic: Method overloading
  • __toString: String conversion
  • __invoke: Make object callable
  • __clone: Object cloning
  • __sleep, __wakeup: Serialization
  • __debugInfo: Debug output customization

9. Sessions

Session Basics

<?php
// Start a session (must be called before any output)
session_start();

// Store data in session
$_SESSION['user_id'] = 123;
$_SESSION['username'] = 'john_doe';
$_SESSION['cart'] = ['item1', 'item2'];

// Access session data
if (isset($_SESSION['user_id'])) {
    echo "User ID: " . $_SESSION['user_id'];
}

// Check if user is logged in
function isLoggedIn(): bool {
    return isset($_SESSION['user_id']);
}

// Remove specific session variable
unset($_SESSION['cart']);

// Destroy entire session
session_unset();     // Clear all session variables
session_destroy();   // Destroy session data

// Note: After session_destroy(), need to start new session
session_start();
?>

Session Security

<?php
// Regenerate session ID to prevent session fixation
session_start();

// Regenerate ID on login
if ($loginSuccessful) {
    session_regenerate_id(true);  // true = delete old session file
    $_SESSION['user_id'] = $userId;
    $_SESSION['logged_in'] = true;
}

// Secure session configuration
ini_set('session.cookie_httponly', 1);  // Prevent JavaScript access
ini_set('session.cookie_secure', 1);     // HTTPS only
ini_set('session.cookie_samesite', 'Strict');  // CSRF protection
ini_set('session.use_strict_mode', 1);   // Reject uninitialized session ID

// Or set via session_set_cookie_params before session_start()
session_set_cookie_params([
    'lifetime' => 0,           // Until browser closes
    'path' => '/',
    'domain' => 'example.com',
    'secure' => true,          // HTTPS only
    'httponly' => true,        // No JavaScript access
    'samesite' => 'Strict'     // CSRF protection
]);

session_start();

// Session timeout
if (isset($_SESSION['last_activity'])) {
    $timeout = 30 * 60;  // 30 minutes
    if (time() - $_SESSION['last_activity'] > $timeout) {
        session_unset();
        session_destroy();
        header('Location: login.php');
        exit;
    }
}
$_SESSION['last_activity'] = time();
?>
⚠️ Security Checklist:
  • Always regenerate session ID after login
  • Use HTTPS and set cookie_secure flag
  • Set httponly flag to prevent XSS attacks
  • Implement session timeout
  • Use SameSite cookie attribute for CSRF protection

10. Cookies

Setting and Reading Cookies

<?php
// Set a cookie (must be called before any output)
setcookie('username', 'john_doe', time() + 3600, '/');

// Cookie parameters:
// setcookie(name, value, expire, path, domain, secure, httponly)

// Set cookie with all parameters
setcookie(
    'user_token',           // Name
    'abc123xyz',            // Value
    time() + 86400 * 30,    // Expires in 30 days
    '/',                    // Path (entire site)
    'example.com',          // Domain
    true,                   // Secure (HTTPS only)
    true                    // HttpOnly (no JavaScript access)
);

// Read cookie
if (isset($_COOKIE['username'])) {
    $username = $_COOKIE['username'];
    echo "Welcome back, $username!";
}

// Delete cookie (set expiration to past)
setcookie('username', '', time() - 3600, '/');

// Alternative: unset value
setcookie('username', '', 1, '/');
?>

Secure Cookie Configuration

<?php
// PHP 7.3+ array syntax for setcookie
setcookie('secure_cookie', 'value', [
    'expires' => time() + 86400,
    'path' => '/',
    'domain' => 'example.com',
    'secure' => true,
    'httponly' => true,
    'samesite' => 'Strict'  // or 'Lax' or 'None'
]);

// SameSite attribute explained:
// - Strict: Cookie only sent to same site
// - Lax: Cookie sent on top-level navigation
// - None: Cookie sent to all sites (requires Secure flag)

// Remember me functionality
function setRememberMeCookie($userId, $token) {
    $hash = hash_hmac('sha256', $token, SECRET_KEY);
    $value = $userId . ':' . $token . ':' . $hash;
    
    setcookie('remember_me', $value, [
        'expires' => time() + (86400 * 30),  // 30 days
        'path' => '/',
        'secure' => true,
        'httponly' => true,
        'samesite' => 'Strict'
    ]);
}
?>

Cookie vs Session

Feature Cookies Sessions
Storage Location Client-side (browser) Server-side
Storage Limit ~4KB per cookie No practical limit
Expiration Can be permanent Typically temporary
Security Less secure (client-side) More secure (server-side)
Use Case Preferences, tracking Authentication, user data
🔒 Security Warning: Never store sensitive data (passwords, credit cards, personal info) in cookies. Always use secure and httponly flags. Validate and sanitize cookie data before use.

11. Common Pitfalls and Best Practices

Type Comparison Pitfalls

<?php
// Pitfall: Loose comparison surprises
var_dump(0 == "0");        // true
var_dump(0 == "hello");    // true (non-numeric string converts to 0)
var_dump("1" == "01");     // true
var_dump(100 == "1e2");    // true

// Solution: Use strict comparison
var_dump(0 === "0");       // false
var_dump(0 === "hello");   // false

// Pitfall: String "0" is falsy
if ("0") {       // false
if ("") {        // false
if ("hello") {   // true

// Solution: Explicit checks
if ($str !== "") {
if (strlen($str) > 0) {
?>

Array and Reference Pitfalls

<?php
// Pitfall: Reference in foreach
$arr = [1, 2, 3];
foreach ($arr as &$value) {
    $value *= 2;
}
// $value still references last element!
foreach ($arr as $value) {
    // This modifies the array unexpectedly
}

// Solution: Unset reference after loop
unset($value);

// Pitfall: Array + operator vs array_merge
$arr1 = ['a' => 1, 'b' => 2];
$arr2 = ['b' => 3, 'c' => 4];
$result = $arr1 + $arr2;  // ['a' => 1, 'b' => 2, 'c' => 4]
// array_merge would give: ['a' => 1, 'b' => 3, 'c' => 4]

// Pitfall: count() on non-countable
$var = null;
count($var);  // Warning in PHP 7.2+, use is_countable()
?>

Function and Scope Pitfalls

<?php
// Pitfall: Variable scope in functions
$x = 10;
function test() {
    echo $x;  // Undefined variable (local scope)
}

// Solution: Use global or pass as parameter
function test() {
    global $x;
    echo $x;  // 10
}

// Pitfall: Headers already sent
echo "Hello";
setcookie('name', 'value');  // Error: headers already sent

// Solution: Set cookies/headers before any output
// Or use output buffering: ob_start()

// Pitfall: Include/require in loops
for ($i = 0; $i < 10; $i++) {
    include 'config.php';  // Redefines functions/classes
}

// Solution: Use include_once or require_once
?>

Security Pitfalls

<?php
// Pitfall: SQL injection
$query = "SELECT * FROM users WHERE id = {$_GET['id']}";

// Solution: Use prepared statements
$stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([$_GET['id']]);

// Pitfall: XSS vulnerability
echo "Hello " . $_GET['name'];  // Dangerous!

// Solution: Escape output
echo "Hello " . htmlspecialchars($_GET['name'], ENT_QUOTES, 'UTF-8');

// Pitfall: Using extract() on user input
extract($_POST);  // Overwrites variables!

// Solution: Access $_POST directly
$username = $_POST['username'] ?? '';
?>

Best Practices Summary

✅ Key Best Practices:
  • Always use strict comparison (===) unless you need type juggling
  • Unset references after foreach loops
  • Use prepared statements for database queries
  • Escape output with htmlspecialchars() for XSS prevention
  • Validate and sanitize all user input
  • Use type hints and return type declarations
  • Enable error reporting in development, log errors in production
  • Use password_hash() and password_verify() for passwords
  • Set secure cookie and session configurations
  • Keep PHP and dependencies up to date
